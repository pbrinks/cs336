Lab 3

Answers for the lab 3 excercises:

  • 3.1
    • Compare and contrast the purposes of Node.js and of Express.js.
      • Node.js is used for building any server-side event-driven i/o application using JavaScript, while Express.js is a framework that goes on top of Node.js that can only be used for web developement.
    • Explain the purpose of the packages.json file.
      • The packages.json file contains metadata about your app, which allows npm to identify the app and handle it's dependencies.
  • 3.2
    • Explain the sense in which these files are “static”.
      • They are static because you cannot change them in the code, however, they can be changed elsewhere.
    • Determine whether the server continues to serve up the hello-world message.
      • Yes it does, but my webpage is showing over top of it.
